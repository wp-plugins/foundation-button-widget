Add Foundation buttons to your sidebars through widgets.

== Description ==

If you have Foundation running on your website and you would like to allow buttons to be created in your theme then allow users to add any Foundation style button through the WordPress widgets menu.

== Installation ==

1. Upload the `admire-foundation-button-widget` folder to the `/wp-content/plugins/` directory, or search for **Foundation Button Widget** in your WordPress admin area.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Once activated you will Foundation buttons availble in your widgets menu. Simply, drag and drop into your sidebar areas.
